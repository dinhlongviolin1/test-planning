# UX Guidelines

## Context Page

- Sidebar: folder tree on left, doc editor on right
- Background polling: every 5s to pick up AI agent changes
- "Import from Repo" button: recursive folder + file sync
- "Add to Context" button on uploads: converts OCR output to PM doc

## Issue Tracker

- Sidebar shows repos under active project when viewing an issue
- Click repo → filters issues by `repo_full_name`
- Click project → clears filter, shows all issues

## Tiptap Editor

- Saves as Markdown (not Tiptap JSON)
- Press `/` for block commands
- Press `Space` for AI inline suggestions (if configured)
