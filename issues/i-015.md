## Title
Fix CLI curl bash install issue - Illegal instruction crash

## Description
Users running the curl-to-bash install script encounter an "Illegal instruction" crash immediately after installation when the installed `tokamak` binary attempts to execute. The crash occurs because the release binary is compiled with AVX2 or AVX-512 CPU extensions that are not available on the user's CPU. The install script must detect CPU capabilities at install time and download the appropriate pre-built binary variant so the binary can actually execute on the target machine.
